# お菓子なSNS 🍩

全国のお菓子好きが、写真付きで自分のお気に入りのお菓子を投稿・共有することができるSNSアプリケーションです。

**🔗 デモサイト（実際に動かせます）**
[https://okashina-sns.bolt.host/](https://okashina-sns.bolt.host/)

## 📖 概要
「地域限定の珍しいお菓子」や「隠れた名作お菓子」の情報をストックし、共有するためのプラットフォームです。
一般的なSNSでは埋もれてしまいがちなローカルなお菓子の情報を、写真とタグ付けで整理し、後からでも探しやすくしました。

### 💡 解決したい課題
テレビやSNSで「このお菓子おいしそう！」と思っても、タイムラインの流れで情報はすぐに埋もれてしまい、「あとから探せない」「商品名を忘れてしまった」という経験がよくあります。
お菓子に特化したこのSNSでは、閲覧者が「好きなお菓子を探す楽しさ」を感じられ、投稿者も気軽に情報を発信できる場所を提供します。

## ✨ 主な機能とこだわり

* **写真・タグ付け必須の投稿**
    * テキストだけでなく、ビジュアルと検索性（タグ）を重視しています。
* **「コメント機能なし」の平和な設計**
    * SNS特有の「荒れる」要素を排除するため、コメント機能はあえて実装していません。
    * 代わりに複数種の「リアクションスタンプ」を用意し、ポジティブな感情のみを共有できます。
* **地域・カテゴリー検索**
    * 旅行先のお土産選びや、自分へのご褒美探しに役立ちます。

## 🛠️ 技術スタック・開発手法
本アプリケーションは、生成AIを活用した現代的な開発手法「**Vibe Coding**」によって構築されました。

* **開発ツール**: [bolt.new](https://bolt.new/)
* **開発手法**: 自然言語によるプロンプトエンジニアリング（コーディングはAIが主導）
* **Frontend**: React, Vite, Tailwind CSS
* **Backend/Database**: Supabase
* **Icons**: Lucide React

## ⚠️ ご利用上の注意（重要）

**1. アカウント作成について**
本アプリはポートフォリオ用のデモ環境です。セキュリティ保護のため、以下のルールを守ってください。
* **普段使用しているメールアドレス・パスワードは絶対に入力しないでください。**
* メール認証はありません。`test@test.com` のような架空のアドレス形式でログイン可能です。

**2. 投稿画像について**
* **お菓子の画像**のみを投稿するよう配慮をお願いします。
* 関係のない画像が投稿された場合、運営側で削除する場合があります。
